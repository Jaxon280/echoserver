Echo servers in some languages
=================================

sample

いろんな言語＋フレームワークで echo server を書いてみました。

いくつかのコードはサンプルプログラムにあったそのままのコードを書いていて、
epoll, thread 版は自分で作成したものです。

各 echo server の実装は全く同じものではないのであくまでも参考ですが、
パフォーマンスはこんなかんじです。
